# Teletype to EPUB Converter

Утилита для автоматического скачивания глав с [Teletype.in](https://teletype.in) и сборки их в электронную книгу формата EPUB. 

Скрипт разработан для сохранения веб-новелл, статей и блогов в удобный формат для чтения на электронных книгах (Kindle, PocketBook, Onyx Boox и др.).

##  Возможности

*   **Интерактивное меню**: Удобный ввод данных через консоль.
*   **Умный парсинг**: Поддержка нескольких источников (редакторов/авторов) для одной главы.
*   **Система приоритетов**: Вы сами выбираете, чью версию главы скачивать в первую очередь (например, если перевод ведут несколько групп). 
*   **Поддержка изображений**: Возможность включить или выключить скачивание картинок.
*   **Кэширование**: Скрипт сохраняет скачанные главы локально. При повторном запуске (например, если интернет прервался) уже скачанные главы берутся из кэша.
*   **Обложки**: Поддержка добавления собственной обложки для книги.

##  Требования

Для работы скрипта необходимо установить:

1.  [Python 3.8+](https://www.python.org/downloads/)
2.  Google Chrome (браузер должен быть установлен в системе).

##  Установка

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/ВАШ_НИКНЕЙМ/teletype-to-epub.git
    cd teletype-to-epub
    ```

2.  **Создайте виртуальное окружение (рекомендуется):**
    *   *Linux/macOS:*
        ```bash
        python3 -m venv venv
        source venv/bin/activate
        ```
    *   *Windows:*
        ```powershell
        python -m venv venv
        venv\Scripts\activate
        ```

3.  **Установите зависимости:**
    ```bash
    pip install -r requirements.txt
    ```

##  Использование

### 1. Подготовка файла со ссылками
Создайте файл `links.txt` (или используйте `example.txt`), куда поместите ссылки на главы. формат должен быть таким, чтобы в строке встречался номер главы и ссылка.

Пример содержимого `links.txt`:
```text
Глава 1 (https://teletype.in/@author1/chapter1)
Глава 1 (https://teletype.in/@author2/backup_chapter1)

Глава 2 (https://teletype.in/@author1/chapter2)
```

### 2. Запуск
Запустите скрипт командой:

```bash
python build_epub.py
```

Следуйте инструкциям на экране:
1.  Введите название книги и автора.
2.  Укажите файл со ссылками.
3.  Расставьте приоритет редакторов (скрипт сам найдет всех авторов из ссылок).
4.  Выберите диапазон глав.
5.  Дождитесь окончания парсинга.

Файл книги (`Название.epub`) появится в корневой папке проекта.

##  Технические детали

*   **Движок**: Используется Selenium WebDriver для корректной отрисовки страниц Teletype (JS-рендеринг).
*   **Драйвер**: Скрипт использует `webdriver-manager` (встроенный в Selenium 4.6+), поэтому ручная установка `chromedriver` не требуется.
*   **Кроссплатформенность**: Работает на Linux, macOS и Windows.
